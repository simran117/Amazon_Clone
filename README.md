## 📌 About The Project

Amazon Clone is a frontend web development project inspired by the Amazon shopping website.

The project focuses on creating a modern e-commerce user interface with a structured layout, attractive design, and responsive web elements using HTML and CSS.


## 🚀 Features

✨ Amazon-style navigation bar  
✨ Product showcase section  
✨ Category cards  
✨ Shopping interface design  
✨ Responsive webpage layout  
✨ Clean and modern UI  
✨ Footer section with useful links  


## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Website structure |
| CSS3 | Styling and layout design |
